Unified image toolkit: resize, convert formats, generate with DALL-E, analyze, and extract text (OCR).

Overview

cc-image is a unified command-line toolkit for image operations. It combines local processing (resize, convert) with AI-powered features (describe, OCR, generate) in a single tool.

API Note: The info, resize, and convert commands work offline. The describe, ocr, and generate commands require an OpenAI API key.

Features

  • info - Display image metadata (dimensions, format, size)
  • resize - Resize images with aspect ratio preservation
  • convert - Convert between formats (PNG, JPEG, WebP, etc.)
  • describe - Get AI description of image content
  • ocr - Extract text from images
  • generate - Create images with DALL-E

Commands

General Syntax
cc-image COMMAND [ARGUMENTS] [OPTIONS]

info - Image Metadata

Display image dimensions, format, color mode, and file size.

Usage
cc-image info IMAGE_FILE
Example
cc-image info photo.jpg

# Output:
# Dimensions: 1920 x 1080
# Format: JPEG
# Mode: RGB
# Size: 245.3 KB

resize - Resize Images

Resize images while preserving aspect ratio. Specify width, height, or both.

Option Description
-o, --output Output file path (required)
-w, --width Target width in pixels
-h, --height Target height in pixels
-q, --quality JPEG quality 1-100 (default: 95)
Examples
# Resize to 800px width (height auto-calculated)
cc-image resize photo.jpg -o thumb.jpg -w 800

# Resize to specific height
cc-image resize photo.jpg -o small.jpg -h 600

# Both dimensions (may not preserve aspect ratio)
cc-image resize photo.jpg -o resized.jpg -w 1024 -h 768

convert - Format Conversion

Convert images between formats. Output format is determined by file extension.

Option Description
-o, --output Output file path (required)
-q, --quality JPEG quality 1-100 (default: 95)
Examples
# PNG to JPEG
cc-image convert screenshot.png -o screenshot.jpg

# JPEG to WebP
cc-image convert photo.jpg -o photo.webp

# Convert with quality setting
cc-image convert image.png -o image.jpg -q 85

describe - AI Image Analysis

Get a detailed AI-generated description of image content using GPT-4 Vision.

Requires: OPENAI_API_KEY environment variable
Usage
cc-image describe IMAGE_FILE
Example
cc-image describe photo.jpg

# Output:
# This image shows a sunset over a mountain range...
# The sky displays vibrant orange and purple hues...

ocr - Extract Text

Extract text from images using GPT-4 Vision. Works with screenshots, documents, signs, and more.

Usage
cc-image ocr IMAGE_FILE
Example
cc-image ocr receipt.jpg

# Output:
# GROCERY STORE
# Date: 2024-01-15
# Apples $3.99
# Bread $2.49
# Total $6.48

generate - Create with DALL-E

Generate images from text descriptions using DALL-E 3.

Option Default Description
-o, --output Required Output file path
-s, --size 1024x1024 Size: 1024x1024, 1024x1792, 1792x1024
-q, --quality standard Quality: standard, hd
Examples
# Generate square image
cc-image generate "A cat wearing a space helmet" -o space_cat.png

# Portrait orientation
cc-image generate "Mountain landscape at sunset" -o mountains.png -s 1024x1792

# High quality
cc-image generate "Detailed steampunk clockwork" -o clock.png -q hd

How It Works

cc-image combines local Pillow-based operations with OpenAI API-powered features

Local Processing

The info, resize, and convert commands use Pillow (PIL) for fast, local image processing without any external API calls.

AI-Powered Features

The describe and ocr commands send images to GPT-4 Vision for analysis. The generate command uses DALL-E 3 to create images from text prompts.
