cc-whisper

Transcribe audio files to text using OpenAI Whisper with optional timestamps.

Overview

cc-whisper is a simple audio transcription tool powered by OpenAI's Whisper API. It converts audio files to text with optional word-level timestamps and supports multiple languages.

Requires: OpenAI API key set as OPENAI_API_KEY environment variable.

Features

  • Transcribe any audio format (mp3, wav, m4a, flac, etc.)
  • Optional word-level timestamps
  • Multi-language support with auto-detection
  • Output to console, file, or JSON
  • Simple, focused interface
Audio Only: For video files with screenshot extraction, use cc-transcribe instead.

Requirements

Set API Key
# Windows
set OPENAI_API_KEY=your-key-here

# macOS / Linux
export OPENAI_API_KEY=your-key-here

Usage

Basic Syntax
cc-whisper AUDIO_FILE [OPTIONS]

Options

Option Default Description
-o, --output None Output file (default: print to console)
-l, --language Auto Language code (en, es, de, fr, etc.)
-t, --timestamps false Include timestamps in output
--json false Output as JSON format
-v, --version - Show version and exit

Supported Languages

Whisper supports many languages. Common codes:

Code Language Code Language
en English fr French
es Spanish de German
zh Chinese ja Japanese
ko Korean pt Portuguese

Output Formats

Plain Text (Default)

Without timestamps
Hello and welcome to today's meeting. We'll be discussing the quarterly results and our plans for next year.

With Timestamps

Using -t flag
[00:00] Hello and welcome to today's meeting.
[00:04] We'll be discussing the quarterly results
[00:08] and our plans for next year.

JSON Format

Using --json flag
{
"text": "Hello and welcome...",
"segments": [
{"start": 0.0, "end": 3.5, "text": "Hello and welcome..."},
{"start": 3.5, "end": 7.2, "text": "We'll be discussing..."},
...
],
"language": "en"
}

Examples

Simple Transcription

Print to console
cc-whisper recording.mp3

Save to File

Output to text file
cc-whisper interview.m4a -o transcript.txt

With Timestamps

Include time markers
cc-whisper podcast.mp3 -o transcript.txt -t

Specific Language

Force Spanish detection
cc-whisper audio_es.wav -l es -o spanish.txt

JSON Output

Full JSON with segments
cc-whisper meeting.mp3 -o result.json --json

How It Works

cc-whisper sends audio directly to OpenAI Whisper API and formats the response

Processing Steps

  1. Load Audio: Reads the audio file from disk
  2. Upload to API: Sends audio to OpenAI Whisper endpoint
  3. Receive Transcription: Gets text with optional segment timestamps
  4. Format Output: Outputs as plain text, timestamped text, or JSON

cc-whisper vs cc-transcribe

Feature cc-whisper cc-transcribe
