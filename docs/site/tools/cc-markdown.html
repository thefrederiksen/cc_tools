Overview

cc-markdown is a command-line tool that converts Markdown files to PDF, Word (.docx), and HTML formats. It includes 7 built-in professional themes and supports custom CSS styling.

No API Required: cc-markdown works entirely offline. No external services or API keys needed.

Features

  • Convert Markdown to PDF, Word, and HTML
  • 7 built-in professional themes
  • Custom CSS support
  • GitHub Flavored Markdown (tables, task lists, code blocks)
  • Configurable page size and margins
  • Single executable - no dependencies

Installation

Download the executable for your platform from GitHub Releases:

Download
# Windows
curl -LO https://github.com/CenterConsulting/cc-tools/releases/latest/download/cc-markdown.exe

# macOS / Linux
curl -LO https://github.com/CenterConsulting/cc-tools/releases/latest/download/cc-markdown
chmod +x cc-markdown

Usage

Basic Syntax
cc-markdown INPUT_FILE -o OUTPUT_FILE [OPTIONS]

The output format is automatically detected from the file extension (.pdf, .docx, or .html).

Options

Option Default Description
-o, --output Required Output file path (.pdf, .docx, or .html)
-t, --theme paper Built-in theme name
--css None Custom CSS file path
--page-size a4 Page size for PDF (a4, letter)
--margin 1in Page margin for PDF
--themes - List available themes and exit
-v, --version - Show version and exit

Themes

cc-markdown includes 7 professionally designed themes:

Theme Description
paper Minimal, clean, elegant (default)
boardroom Corporate, executive style with serif fonts
terminal Technical, monospace with dark-friendly colors
spark Creative, colorful, modern
thesis Academic, scholarly with proper citations style
obsidian Dark theme with subtle highlights
blueprint Technical documentation style

Examples

Convert to PDF

PDF with default theme
cc-markdown report.md -o report.pdf

Convert with Theme

PDF with boardroom theme
cc-markdown proposal.md -o proposal.pdf --theme boardroom

Convert to Word

Word document
cc-markdown notes.md -o notes.docx --theme thesis

Custom CSS

With custom styling
cc-markdown document.md -o document.pdf --css my-styles.css

Custom Page Size

Letter size with narrow margins
cc-markdown doc.md -o doc.pdf --page-size letter --margin 0.75in

How It Works

cc-markdown parses Markdown, applies theme styling, and converts to the target format

Processing Steps

  1. Parse Markdown: Reads and parses the input file using a GFM-compatible parser
  2. Load Theme: Loads the selected built-in theme or custom CSS
  3. Generate HTML: Converts parsed content to styled HTML
  4. Convert Output: Renders to PDF (via headless Chrome), Word (via python-docx), or outputs HTML directly
