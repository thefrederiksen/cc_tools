How CC Tools Work

Simple CLI interfaces that integrate seamlessly with AI coding assistants

AI assistants call CC Tools directly via command line to transform files and generate outputs

Available Tools

Each tool is a standalone executable - download and run

cc-markdown

Convert Markdown to beautifully styled PDF, Word, and HTML documents with built-in professional themes.

PDF Word HTML 7 Themes No API

cc-transcribe

Transcribe videos with timestamps and automatically extract screenshots at content changes.

Video Input Timestamps Screenshots OpenAI API

cc-image

Unified image toolkit: resize, convert formats, generate with DALL-E, analyze, and extract text (OCR).

Resize Convert Generate OCR Describe

cc-voice

Convert text to natural-sounding speech with multiple voices, automatic markdown cleaning, and long text support.

6 Voices HD Quality Markdown Clean OpenAI API

cc-whisper

Transcribe audio files to text using OpenAI Whisper with optional word-level timestamps.

Audio Input Timestamps Multi-language OpenAI API

cc-video

Video utilities: get info, extract audio tracks, capture screenshots at content changes or specific timestamps.

Video Info Extract Audio Screenshots No API

Requirements

What you need to run CC Tools

OpenAI API Key

Tools that use AI features (transcription, image generation, TTS) require an OpenAI API key:

Linux / macOS
export OPENAI_API_KEY=your-key-here
Windows
set OPENAI_API_KEY=your-key-here
No API needed: cc-markdown and cc-video work entirely offline without an API key.

FFmpeg (for video tools)

cc-video and cc-transcribe require FFmpeg for video processing:

Install FFmpeg
# Windows (Chocolatey)
choco install ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Linux (apt)
sudo apt install ffmpeg
