Getting Started

Download, configure, and start using CC Tools in minutes

Download

CC Tools are distributed as standalone executables. Download the tools you need from GitHub Releases:

GitHub Releases
https://github.com/CenterConsulting/cc-tools/releases

Each release includes executables for:

  • Windows: .exe files
  • macOS: Universal binaries
  • Linux: x64 binaries

Installation

No installation required - just download and run. Optionally, add the tools to your PATH:

Windows (PowerShell)
# Create a tools directory
mkdir C:\tools

# Add to PATH (run as administrator)
$path = [Environment]::GetEnvironmentVariable("PATH", "Machine")
[Environment]::SetEnvironmentVariable("PATH", "$path;C:\tools", "Machine")

# Copy tools to directory
copy cc-markdown.exe C:\tools\
macOS / Linux
# Move to /usr/local/bin (requires sudo)
sudo mv cc-markdown /usr/local/bin/
sudo chmod +x /usr/local/bin/cc-markdown

# Or add to your own bin directory
mkdir -p ~/bin
mv cc-markdown ~/bin/
chmod +x ~/bin/cc-markdown
# Add ~/bin to PATH in .bashrc or .zshrc

Requirements

OpenAI API Key

Some tools require an OpenAI API key for AI features (transcription, TTS, image generation, vision):

Tool API Required Features Using API
cc-markdown No -
cc-transcribe Yes Whisper transcription
cc-image Partial describe, ocr, generate
cc-voice Yes Text-to-speech
cc-whisper Yes Audio transcription
cc-video No -

Set your API key as an environment variable:

Windows (Command Prompt)
set OPENAI_API_KEY=sk-your-key-here
Windows (PowerShell - Permanent)
[Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-your-key-here", "User")
macOS / Linux
# Temporary (current session)
export OPENAI_API_KEY=sk-your-key-here

# Permanent (add to ~/.bashrc or ~/.zshrc)
echo 'export OPENAI_API_KEY=sk-your-key-here' >> ~/.bashrc

FFmpeg

cc-video and cc-transcribe require FFmpeg for video processing:

Install FFmpeg
# Windows (Chocolatey)
choco install ffmpeg

# Windows (Scoop)
scoop install ffmpeg

# macOS (Homebrew)
brew install ffmpeg

# Linux (apt)
sudo apt install ffmpeg

# Linux (dnf)
sudo dnf install ffmpeg

Quick Start Examples

Convert Markdown to PDF

cc-markdown
cc-markdown report.md -o report.pdf --theme boardroom

Transcribe a Video

cc-transcribe
cc-transcribe lecture.mp4 -o ./transcript

Resize an Image

cc-image
cc-image resize photo.jpg -o thumb.jpg -w 800

Text to Speech

cc-voice
cc-voice "Hello, welcome to CC Tools!" -o greeting.mp3 -v nova

Transcribe Audio

cc-whisper
cc-whisper meeting.mp3 -o transcript.txt -t

Extract Video Screenshots

cc-video
cc-video screenshots presentation.mp4 -o ./slides

Using with AI Assistants

CC Tools are designed for use by AI coding assistants (Claude, GPT, etc.). The tools follow consistent patterns:

  • Clear exit codes: 0 for success, non-zero for errors
  • Structured output: Consistent format for parsing
  • File-based I/O: Input and output via files, not stdin/stdout
  • Helpful errors: Descriptive error messages with suggested fixes
MCP Integration: CC Tools can be exposed as MCP (Model Context Protocol) tools for seamless integration with Claude and other AI assistants.
